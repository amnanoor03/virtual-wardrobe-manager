Virtual Wardrobe Manager
A terminal-based wardrobe management system built in Python using Object-Oriented Programming. Features 4 classes (ClothingItem, Wardrobe, Outfit, FashionCollection) with full CRUD operations, dynamic search using **kwargs, and outfit validation logic.
Tech: Python 3 | OOP | Terminal UI
Run: python Fashion.py
