# Virtual Wardrobe Manager

A terminal-based wardrobe management system built in Python using Object-Oriented Programming principles.

## Features
- Add, remove, and search clothing items by multiple attributes
- Create outfits with validation logic (must include top, bottom, and footwear)
- Organize complete outfits into a fashion collection
- Dynamic multi-attribute search using `**kwargs`

## Tech Stack
- Python 3
- OOP (4 classes: ClothingItem, Wardrobe, Outfit, FashionCollection)
- Terminal UI

## How to Run
1. Clone the repo
2. Run: `python Fashion.py`

## Project Structure
- `Fashion.py` — main application with all classes and terminal interface
